A full desktop in your browser

A self-hosted cloud desktop with shell, terminal, file manager, AI, and remote access — no install required.

▦

Desktop Shell

Window manager with drag, resize, snap, multi-desktop, dock, and launchpad.

〉

Terminal

Persistent PTY sessions. Disconnect and reattach without losing work.

⚙

AI Assistant

Ollama, OpenAI, and Anthropic. Sandboxed code execution.

⊙

Chromium

Remote browser streamed via WebRTC with full input support.

📁

File Manager

Browse, upload, download, rename, and manage files.

🔒

Auth & Security

Multi-user, OAuth, sessions, rate limiting, privilege separation.

📦

App Hub

Install apps from the registry with isolated execution.

📡

Remote Desktop

Built-in tunneling for remote desktop access to your cloud OS from anywhere.

Secure login

Multi-user authentication with password-synced sudo. Create accounts, manage profiles, sign in with OAuth.

Try with Docker

Not ready to install? Run in a container first.

# Run Vula OS in Docker
docker run -p 8080:8080 --shm-size=1g \
  -v vulos-data:/root/.vulos \
  ghcr.io/vul-os/vulos:latest

Then open {{APP_URL}}
