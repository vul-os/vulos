A full OS in your browser

Desktop shell, terminal, file manager, AI assistant, and more — built on Alpine Linux.

▦

Desktop Shell

Window manager with drag, resize, snap, multi-desktop, dock, and launchpad. Works on desktop and mobile.

〉

Terminal

Full PTY terminal with session persistence. Disconnect and reattach without losing your work.

⚙

AI Assistant

Pluggable AI with Ollama, OpenAI, and Anthropic backends. Code generation with sandboxed Python execution.

⊙

Chromium Browser

Remote browser streamed via WebRTC. Browse the web from inside your OS with full keyboard and mouse support.

📁

File Manager

Browse, upload, download, and manage files. Create directories, rename, delete, and navigate freely.

🔒

Auth & Security

Multi-user with profiles, OAuth support, session management, rate limiting, and privilege separation.

📦

App Hub

Install and manage apps from the registry. Python plugin system with JSON manifests and isolated execution.

📡

Remote Access

Built-in tunneling for accessing your OS from anywhere. Expose apps via public URLs.

Get Vula OS

Run with Docker in one command, or pull the image for your platform.

Quick Start

# Run Vula OS
docker run -p 8080:8080 --shm-size=1g \
  -v vulos-data:/root/.vulos \
  ghcr.io/vul-os/vulos:latest

Then open http://localhost:8080
